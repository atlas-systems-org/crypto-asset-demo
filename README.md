# Crypto Asset Demo

Sample application demonstrating various cryptographic asset patterns across a typical microservice architecture.

This repository is used to demonstrate Qryptive's Cryptographic Asset Discovery Engine.

## Structure

```
src/auth/          - Authentication service with crypto utilities
config/            - Application and infrastructure configuration
certs/             - TLS certificates and keys
infra/             - Terraform infrastructure definitions
deploy/            - CI/CD pipeline definitions
```
